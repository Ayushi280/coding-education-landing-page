# Coding Education Landing Page - Coding Monkeys

A responsive and modern landing page built using **HTML**, **CSS**, and **Bootstrap 5**. This project showcases an online coding education platform with a clean and user-friendly interface.

## Features

- Responsive Bootstrap 5 design
- Interactive image carousel , use of cards , forms.
- Sections for Web Development, Data Science, Machine Learning, Cloud Computing, and Cyber Security
- Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
